# Data Source Classification

## Official/Given Data

### From C-MAPSS (NASA Turbojet Dataset)
- **train.csv** ✅ Official
  - Raw sensor data for training
  - Columns: EngineID, Cycle, Altitude, Mach, Tamb, Pamb, RPM, FuelFlow, Pressures, Temperatures
  
- **test.csv** ✅ Official
  - Raw sensor data for testing
  - Same columns as train.csv
  
- **turbojet_complete_dataset.csv** ✅ Official + Generated
  - Complete dataset with derived health metrics
  - Adds: CompressorHealth, CombustorHealth, TurbineHealth, OverallHealth, Thrust, TSFC
  - **Hybrid:** Base data from C-MAPSS, health metrics calculated by your model
